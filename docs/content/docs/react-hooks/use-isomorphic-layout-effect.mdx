---
title: useIsomorphicLayoutEffect
---

`useLayoutEffect`와 같은 시그니처이지만, **브라우저가 아닌 환경(SSR)** 에서는 `useEffect`로 대체합니다.  
SSR에서 `useLayoutEffect` 경고를 피하고 클라이언트에서는 레이아웃 직후 동기 이펙트가 필요할 때 사용해 보세요.

## Parameters

React의 `useLayoutEffect`와 동일합니다.

| 이름     | 타입             | 설명                                                                                                 |
| -------- | ---------------- | ---------------------------------------------------------------------------------------------------- |
| `effect` | `EffectCallback` | 브라우저가 화면을 그린 뒤·DOM 업데이트 반영 직후 실행할 이펙트 함수. 정리 함수를 반환할 수 있습니다. |
| `deps`   | `DependencyList` | (선택) 의존성 배열. 생략 시 매 렌더마다 이펙트가 실행될 수 있습니다.                                 |

## Response

반환값 없음 (`void`)

## Example

```tsx
import { useIsomorphicLayoutEffect } from '@maru-ui/react-hooks';

useIsomorphicLayoutEffect(() => {
  // 작업 수행
}, [deps]);
```
