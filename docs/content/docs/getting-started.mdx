## 제공하는 라이브러리

| 패키지                 | 역할                                                                           |
| ---------------------- | ------------------------------------------------------------------------------ |
| `@maru-ui/core`        | 프레임워크에 묶이지 않는 공통 로직·엔진. 컴포넌트와 훅이 함께 쓰는 기반입니다. |
| `@maru-ui/react-hooks` | Maru UI 전반에서 재사용하는 React 훅 모음.                                     |
| `@maru-ui/picker-view` | 휠/스크롤형 선택 UI와, 같은 동작을 **Headless**로 쓰기 위한 훅.                |

## 빠른 시작: PickerView

### 설치

#### pnpm

```bash
pnpm add @maru-ui/picker-view
```

#### npm

```bash
npm i @maru-ui/picker-view
```

#### yarn

```bash
yarn add @maru-ui/picker-view
```
