# Picker View

선택형 리스트를 위한 컴포넌트와 Headless 훅입니다.

## PickerView (UI)

기본 UI가 포함된 컴포넌트입니다.

```tsx
import { PickerView } from '@maru-ui/picker-view';
import '@maru-ui/picker-view/styles';

<PickerView items={['옵션 1', '옵션 2', '옵션 3']} visibleCount={5} />;
```

## usePickerView (Hook)

커스텀 UI를 만들 때 사용하는 훅입니다.

```tsx
import { usePickerView } from '@maru-ui/picker-view/hook';

const { selectedIndex, setSelectedIndex, visibleRange } = usePickerView({
  itemCount: 10,
  initialIndex: 0,
  visibleCount: 5,
});
```

## API

### UsePickerViewOptions

| Prop           | Type     | Default | 설명                       |
| -------------- | -------- | ------- | -------------------------- |
| `itemCount`    | `number` | -       | 아이템 개수                |
| `initialIndex` | `number` | `0`     | 초기 선택 인덱스           |
| `visibleCount` | `number` | `5`     | 한 번에 보여줄 아이템 개수 |

### UsePickerViewReturn

| 값                 | Type                      | 설명                         |
| ------------------ | ------------------------- | ---------------------------- |
| `selectedIndex`    | `number`                  | 현재 선택된 인덱스           |
| `setSelectedIndex` | `(index: number) => void` | 인덱스 설정                  |
| `visibleRange`     | `[number, number]`        | 뷰포트 내 보이는 인덱스 범위 |
